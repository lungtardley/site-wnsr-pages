# site-wnsr-pages

本仓库用于归档与发布多个独立的 HTML 页面。这些页面通常作为静态资源被托管，不针对任何特定域名或网站，仅用于演示、测试或存档用途。

## 项目简介

这是一个轻量级的 HTML 页面集合仓库。每个页面独立存在，不依赖后端服务或数据库，适合直接通过 GitHub Pages 或其他静态托管服务发布。仓库中的页面内容涵盖多种用途，如工具页面、信息展示页、临时演示页等。

## 目录说明

```
.
├── index.html          # 仓库首页或导航页（如有）
├── page1.html          # 示例页面 1
├── page2.html          # 示例页面 2
├── assets/             # 静态资源文件夹（CSS、JS、图片等）
│   ├── style.css
│   └── script.js
└── README.md           # 本文件
```

目录结构可能随页面增加而调整。每个页面通常自包含样式与脚本，以减少外部依赖。

## 页面归档说明

- 每个 HTML 页面均保持独立，不互相依赖。
- 页面可能包含外部资源（如字体、图标库），但尽量使用 CDN 或本地资源。
- 部分页面为历史版本存档，内容可能不再更新，仅作参考。
- 页面命名建议保持简洁且具描述性，避免使用特殊字符。

## 维护说明

- 欢迎提交 Pull Request 添加新页面或修正现有页面。
- 新页面请放置在根目录或合适的子目录中，并确保路径引用正确。
- 请勿在页面中包含恶意代码、跟踪脚本或违反平台政策的内容。
- 仓库维护者会不定期清理不再需要的页面或资源。

## 使用方式

1. 克隆本仓库到本地。
2. 直接在浏览器中打开任意 `.html` 文件即可预览。
3. 如需部署到 GitHub Pages，请在仓库设置中开启 Pages 功能，并选择 `main` 分支的根目录作为发布源。
